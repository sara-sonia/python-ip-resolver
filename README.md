# Python IP Resolver

This project is a simple Python script that uses the socket module to resolve a hostname into an IP address. The script demonstrates basic networking functionality and how domain names are translated into numerical IP addresses.

## How to run

1. Clone the repository:
```bash
git clone https://github.com/sara-sonia/python-ip-resolver.git
```

2. Navigate into the folder:
```bash
cd python-ip-resolver
```

3. Run the script:
```bash
python resolve_ip.py
```

## Requirements
 - Python 3 installed
